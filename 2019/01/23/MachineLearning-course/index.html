Tutorial 01

1 (a) Supervised learning: Discover a mapping between xi and yi (Given xi, say what the correct yi is)

training set：D = {(xi, yi)}i=1N

xi : input data set

yi : output data set

Example : iris dataset, the features are measurements that describe each flower(petal area and sepal area), and the task is to identify which species a given Iris flower comes from.

(b) Unsupervised learning: Look for the patterns in a data set X. A ypatical task is clustering, looking to order data into groups.

training set : D = {(xi)}i=1N

Example : image processing example, elements are pixels, a set of three numbers(R, G, B). The code generate these examples looks to group into clusters pixels that were within some increment of each other.

4 Performance metrics

(1) accuracy = (correct)/(total)

(2) confusion matrix : take predict number 1 to be positive, take 0 to be negative ; actual to be true or false=> TP/FP/TN/FN. (since the large numbers are along the diagonal means mainly classifies correctly)

(3) precision = (TP)/(TP + FP)

(4) recall = (TP)/(TP+FN)

(5) F1score = 2/(1/precision+1/recall)
